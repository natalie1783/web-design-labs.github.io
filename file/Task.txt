Лабораторна робота 2
Тема: Використання графічних об’єктів та гіперпосилань в документах HTML
Мета: набути навичок використання графічних об’єктів та гіперпосилань в документах HTML
Завдання: створити HTML-сторінку відповідної структури та відповідного змісту.

Завдання для виконання роботи
1. Вставити графічне зображення на Web-сторінку
2. Додати фоновий малюнок для Web-сторінки
3. Зробити обрамлення графічного зображення
4. Ввести текст і задати параметри обтікання графіки текстом
5. Вставити нове графічне зображення і вирівняти текст відносно цього малюнку
6. Змінити пусту область навколо графічного зображення
7. Побудувати горизонтальну лінійку і задати її параметри (товщина, довжина, колір)
8. Побудувати маркерний список з графічними маркерами
9. Встановлення інші розміри зображення для другого рисунку
10. Вставити третій малюнок використати зображення з низькою розподільною здатністю
11. Побудувати гіперпосилання, яке вказує на іншу Web-сторінку
12. Побудувати гіперпосилання, яке вказують на Web-сторінку в межах власного сайту
13. Використати засоби навігації для сайту (добавити кнопки)
14. Створити гіперпосилання на об’єкт тієї ж сторінки
15. Створення гіперпосилання на окремі області іншої сторінки
16. Використати графічне зображення, як гіперпосилання
17. Створити гіперпосилання (мініатюрні зображення), які вказують на графічні зображення
18. Створити гіперпосилання, які вказують на файл
19. Створити гіперпосилання на адресу електронної пошти
20. Створити гіперпосилання, яке вказує на сайт FTP
21. При натисканні на гіперпосилання зробити відкриття Web-сторінок в новому вікні браузера
22. Задати колір тексту для всіх типів гіперпосилань
23. Встановити “гарячі клавіші” для гіперпосилання
24. Встановити порядок переходу за гіперпосиланнями
25. Створити гіперпосилання на файл аудіо запису
26. Впровадити звукові об’єкти на Web-сторінку
27. Створити гіперпосилання на файл відеозапису
28. Впровадити об’єкти відео на Web-сторінку













Контрольні питання
Які є способи використання зображення в HTML документі?
Як побудувати гіперпосилання, яке вказують на Web-сторінку в межах власного сайту?
Як створити гіперпосилання на об’єкт однієї сторінки?
Як створити гіперпосилання на адресу електронної пошти?
Як створити, щоб при натисканні на гіперпосилання зробити відкриття Web-сторінок в новому вікні браузера?
